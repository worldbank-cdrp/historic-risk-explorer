## Development

### Dependencies

python

```
pip install -r requirements.txt
```

### Utilities

Mapbox CLI

```
brew install mapbox/cli/mapbox
```

### collecticons

get bower

```
npm install -g bower
```

download collecticons

```
bower install --save collecticons
```

copy collecticons to correct path

```
cp bower_components/collecticons app/asssts/graphics/
```
